Survey API Reference¶

class SurveyOutcomeGenerator(datastore, dataframe=None, clean_folders=False)¶
asset_index(cols, use_weights=True)¶

Derives an asset index - first component of (weighted) PCA - from specified set of columns.

Parameters

  • cols (List[str]) – Columns (i.e. survey questions) to use to derive the index.

  • use_weights (bool) – If true, uses a weighted PCA.

Return type

pandas.core.frame.DataFrame

Returns: Pandas df containing asset index of surveyed users.

fit_pmt(outcome, cols, model_name='linear', kfold=5, use_weights=True, log_transform=False, scale=False, winsorize=False)¶

Trains a model to predict an outcome variable using a set of survey questions - this will be the PMT. Also provides metrics on modelling performance.

Parameters

  • outcome (str) – Target variable.

  • cols (List[str]) – Independent variables.

  • model_name (str) – ML model to use - can be one of [‘linear’, ‘lasso’, ‘ridge’, ‘randomforest’, ‘gradientboosting’]

  • kfold (int) – Number of cross-validation folds to use.

  • use_weights (bool) – If true, uses survey weights.

  • log_transform (bool) – If true, log transforms outcome variable.

  • scale (bool) – If true, standard scales inputs.

  • winsorize (bool) – If ture, winsorizes (.005, .995) inputs.

Return type

pandas.core.frame.DataFrame

Returns: Pandas df of outcome variable, in-sample predictions, and out-of-sample predictions.

pretrained_pmt(other_data_in, cols, model_name, dataset_name='other_data')¶

Use a pre-trained PMT model to predict the outcome variable in another dataset - check that the latter has the same columns and data types of the original survey that was used to train the PMT

Parameters

  • other_data_in (Union[str, pandas.core.frame.DataFrame]) – new dataset to get predictions for, either as path or as pandas df

  • cols (List[str]) – input columns

  • model_name (str) – PMT model to use

  • dataset_name (str) – name of new dataset

Return type

pandas.core.frame.DataFrame

Returns: pandas df of user id and predicted outcome variable

select_features(outcome, cols, n_features, method='correlation', model_name='', kfold=5, use_weights=True, log_transform=False, plot=True)¶

Selects the top-N features that are most useful to predict the outcome variable.

Parameters

  • outcome (str) – Target variable.

  • cols (List[str]) – Independent variables.

  • n_features (int) – Number of features to select.

  • method (str) –

    Method to use to select the top-N features - it can be: ‘correlation’: return top N features most correlated with the outcome ‘lasso’: use LASSO regressions for a grid of penalties to select features, use the one which has the

    closest to n_features features

    OTHER STRING: forward selection with a machine learning model

  • model_name (str) – Name to use when saving outputs.

  • kfold (int) – Number of cross-validation folds to use.

  • use_weights (bool) – If true, uses survey weights.

  • log_transform (bool) – If true, log transforms outcome variable.

  • plot (bool) – If true, plots performance metrics.

Return type

Tuple[List[str], pandas.core.frame.DataFrame]

Returns: Top-N features selected, pandas df of scores.
